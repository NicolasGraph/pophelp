h1. Preferences: Doctype

Select which HTML doctype you wish to use (either HTML5 or XHTML). This affects comment input forms, Textile rendering of <code><abbr></code> or <code><acronym></code>, and <code><link></code> tags generated by <a rel="external" target="_blank" href="http://docs.textpattern.io/tags/css"><code><txp:css /></code></a>.
