h1. Textile: Images

To insert an image, put the URL for the image inside exclamation marks.

*Optional:* text that immediately follows the URL in (parentheses) will be used as the <code>alt</code> text for the image (this also adds a <code>title</code> attribute with the same content). Images on the web should always have descriptive <code>alt</code> text for the benefit of readers using non-graphical browsers.

*Optional:* place a colon followed by a URL immediately after the closing <code>!</code> to make the image into a link.

h3. Examples

bc. !http://textpattern.com/hi.png(Hello World)!

Will become:

bc. <img src="http://textpattern.com/hi.png" alt="Textpattern icon" title="Hello World" />

Which, when rendered by a browser, will appear like this:

!http://textpattern.com/hi.png(Hello World)!

With a link:

bc. !http://textpattern.com/hi.png(Hello World)!:http://textpattern.com

Will become:

bc. <a href="http://textpattern.com"><img src="http://textpattern.com/hi.png" alt="Hello World" title="Hello World" /></a>

Which, when rendered by a browser, will appear like this:

!http://textpattern.com/hi.png(Hello World)!:http://textpattern.com
