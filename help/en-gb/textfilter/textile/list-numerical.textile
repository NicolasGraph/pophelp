h1. Textile: Numerical lists

To make a numerical list, place each item in its own paragraph, preceded by <code>#</code>.

h3. Example

bc. # A first item
# A second item
## A second item first sub item
## A second item second sub item
# A third item
# A fourth item

Will become:

bc. <ol>
  <li>A first item</li>
  <li>A second item
  <ol>
    <li>A second item first sub item</li>
    <li>A second item second sub item</li>
  </ol>
  </li>
  <li>A third item</li>
  <li>A fourth item</li>
</ol>

Which, when rendered by a browser, will appear like this:

# A first item
# A second item
## A second item first sub item
## A second item second sub item
# A third item
# A fourth item

p(alert-block information). *Note:* Unmodified paragraphs will be wrapped in <code><p></code> tags.
