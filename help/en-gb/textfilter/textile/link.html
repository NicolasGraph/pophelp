Textile: Links

To make a hypertext link, put the link text in "quotation marks" followed immediately by a colon and the URL of the link.

Optional: text in (parentheses) following the link text, but before the closing quotation mark, will become a title attribute for the link, visible as a tooltip when a cursor is above it.

Example (standard links)

"This is a link (This is a title)":http://textpattern.com

Will become:

<a href="http://textpattern.com" title="This is a title">This is a link</a>

Which, when rendered by a browser, will appear like this:

This is a link (This is a title)

Example (aliased links)

Textile v2.0.0 onwards also allows storage of long URLs outside of the text youâ€™re working in, and to reference them with aliases of your choosing. Simply store the URL anywhere in the document, such as at the end, using this syntax:

[alias (optional title)]http://url

Each referenced URL must begin on a new line:

This is a "link":example to my website.

So long as this appears elsewhere in the document:

[example]http://example.com/index.html

Will become:

This is a <a href="http://example.com/index.html">link</a> to my website.
