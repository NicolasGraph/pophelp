h1. Txp die

bc. <txp:txp_die />

The *txp_die* tag is a _single_ tag that will terminate normal page rendition and return the given status to the user agent (browser, search engine crawler, feed aggregator). An error page will also be returned to the user agent.

The status can be displayed by the "error_status":http://docs.textpattern.io/tags/error-status tag. A textual message can be associated with the error status and retrieved with the "error_message":http://docs.textpattern.io/tags/error-message tag.

"More information on the *txp_die* tag":http://docs.textpattern.io/tags/txp-die.
