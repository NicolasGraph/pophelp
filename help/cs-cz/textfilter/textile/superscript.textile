h1. Textile: Horní index (angl. "superscript")

Uzavřením textu mezi znaky @ ^@ vypíšete ^superscript^.

Apklikován je HTML tag @<sup>@:

bc. ^fráze^ → <sup>fráze</sup>

Prohlížeč texto text zobrazí jako  ^fráze^.

Stejného efektu jako tagem @<sup>@ se dá dosáhnout pomocí CSS stylu @ vertical-align: super@, ale @<sup>@ je dobrý tag na vyznačení sémantiky (smyslu) textu.
