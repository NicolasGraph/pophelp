h1. Textile: Zvýraznění

Prohlížeče vykreslují kurzívou. Nejsprávnější tag pro zvýrazňování (a pro tento účel jen zřídka používaný).

Okolo zvýrazňovaného textu udělejte ==_podtržítka_==.

Výsledkem je HTML tag @<em>@:

bc. _zvýrazněný text_ → <em>zvýrazněný text</em>

což prohlížeč zobrazí jako _zvýrazněný text_

p(alert-block information). *Pozn.:* text vypsaný kurzívou napíšete i @__obklopením textu__@ __dvojitými podtržítky__, což produkuje html tag <code><i></i></code>, který se používá pro text psaný kurzívou, ale bez významového zvýraznění.
