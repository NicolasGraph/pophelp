Admin-Plugins verwenden

Admin-Plugins sind spezielle Plugins um die Arbeitsweise des Backends anzupassen. Plugins können sich einhängen in bestehende Funktionen, wie das Verfassen oder Bearbeiten von Artikeln, oder sie können über zusätzliche Tabs neue oder erweiterte Funktionalität für das Backend zur Verfügung stellen.

Admin-Plugins erlauben die Erweiterung von Textpattern für viele verschiedene Zwecke. Für mehr Informationen besuchen Sie das "Forum&quot;:http://forum.textpattern.com von Textpattern.
