h1. Fettung

Verwenden Sie ==*Sterne*== um Text *fett zu betonten*.

Folgendes &lt;strong&gt; &lt;acronym title=&quot;Hypertext Markup Language&quot;&gt;HTML&lt;/acronym&gt;-label wird angefügt:

bq. *a phrase* → &lt;code&gt;&lt;strong&gt;a phrase&lt;/strong&gt;&lt;/code&gt; → *a phrase*

Mit zwei Sternen können Sie einen Text lediglich **fetten** (z. B. als Design-Element):

bq. ==**a phrase**== → &lt;code&gt;&lt;b&gt;a phrase&lt;/b&gt;&lt;/code&gt; → **a phrase**

&lt;small&gt;Hinweis: Die Auszeichnungen müssen innerhalb eines Absatzes geöffnet und geschlossen werden um gültigen Code zu erzeugen.&lt;/small&gt;
