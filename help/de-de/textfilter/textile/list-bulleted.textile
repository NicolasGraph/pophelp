h1. Unsortierte Liste

Um eine Aufzählungsliste mit mehreren Punkten zu definieren, schreiben Sie jeden Punkt in eine neue Zeile und setzen Sie “* “ an den Anfang der Zeile.

Beispiel:

==*== Erster Punkt
==*== Zweiter Punkt
==*== Dritter Punkt
==*== Vierter Punkt

wird zu:

@&lt;ul&gt;@
@&lt;li&gt;Erster Punkt&lt;/li&gt;@
@&lt;li&gt;Zweiter Punkt&lt;/li&gt;@
@&lt;li&gt;Dritter Punkt&lt;/li&gt;@
@&lt;li&gt;Vierter Punkt&lt;/li&gt;@
@&lt;/ul&gt;@

Im Browser wird die Aufzählungsliste so dargestellt:

* Erster Punkt
* Zweiter Punkt
* Dritter Punkt
* Vierter Punkt

Hinweis: Unveränderte Absätze werden von @&lt;p&gt;@ tags eingeschlossen.
