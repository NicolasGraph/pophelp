Zitatblock

Um einen Absatz oder eine Zeile als Zitatblock zu formatieren und einzurücken, setzten Sie "bq. " an den Anfang.

Beispiel:

Any old text.

bq. A block quotation.

Any old text.

wird zu:

<p&gt;Any old text.</p&gt;

<blockquote&gt;A block quotation.</blockquote&gt;

<p&gt;Any old text.</p&gt;

Im Browser wird der Text so dargestellt:

Any old text.

A block quotation.

Any old text.

<small&gt;Hinweis: Unveränderte Absätze werden von <p&gt; tags eingeschlossen.</small&gt;
