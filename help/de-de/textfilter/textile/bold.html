Fettung

Verwenden Sie *Sterne* um Text fett zu betonten.

Folgendes <strong&gt; <acronym title="Hypertext Markup Language&quot;>HTML&lt;/acronym&gt;-label wird angefügt:

a phrase → <code&gt;<strong&gt;a phrase&lt;/strong&gt;</code&gt; → a phrase

Mit zwei Sternen können Sie einen Text lediglich fetten (z. B. als Design-Element):

**a phrase** → <code&gt;<b&gt;a phrase&lt;/b&gt;</code&gt; → a phrase

<small&gt;Hinweis: Die Auszeichnungen müssen innerhalb eines Absatzes geöffnet und geschlossen werden um gültigen Code zu erzeugen.</small&gt;
