Les utilisateurs : À propos des privilèges

Rédacteur (ou éditeur)

  • autorisé à gérer les articles, les commentaires et les liens
  • autorisé à utiliser PHP dans les articles si le paramètre d'administration ‘Autoriser PHP dans les articles ?’ est coché sur ‘Oui’
  • autorisé à gérer les images et les fichiers
  • autorisé à gérer les sections et les catégories
  • autorisé à gérer les gabarits (pages), les formulaires et les styles CSS
  • autorisé à consulter les informations de diagnostics
  • autorisé à gérer les préférences (paramètres) du site
  • autorisé à gérer les utilisateurs
  • autorisé à gérer les journaux d’activités
  • autorisé à gérer les plugins
  • autorisé à importer des données

Rédacteur en chef

  • autorisé à gérer les articles, les commentaires et les liens
  • autorisé à utiliser PHP dans les articles si le paramètre d'administration ‘Autoriser PHP dans les articles ?’ est coché sur ‘Oui’
  • autorisé à gérer les images et les fichiers
  • autorisé à gérer les sections et les catégories
  • autorisé à gérer les gabarits (pages), les formulaires et les styles CSS
  • autorisé à consulter les informations de diagnostics
  • autorisé à gérer les préférences du site
  • autorisé à consulter les utilisateurs
  • autorisé à gérer les journaux d’activités
  • autorisé à gérer les plugins
  • autorisé à importer des données

Correcteur

  • peut gérer les articles avec une limitation :
    • n'est pas autorisé à supprimer les articles des autres utilisateurs
  • autorisé à gérer les commentaires et les liens
  • autorisé à gérer les images et les fichiers
  • autorisé à gérer les catégories
  • autorisé à gérer les gabarits (pages) et les formulaires
  • autorisé à consulter les utilisateurs
  • autorisé à gérer les journaux d’activités

Rédacteur simple (ou journaliste)

* autorisé à gérer ses propres articles, consulter ceux des autres utilisateurs

  • autorisé à gérer les images et les fichiers

Freelancer (ou pigiste)

  • autorisé à gérer ses propres articles dans une certaine limite :
    • autorisé à modifier un article uniquement avant sa publication
    • autorisé à changer uniquement les status "Brouillon", "Masqué" ou "À valider"
  • autorisé à consulter les articles des autres utilisateurs

Designer

* Peut gérer ses propres articles dans une certaine limite :

autorisé uniquement à modifier un article dans l'attente d'être publié

autorisé à changer uniquement les status "Brouillon", "Masqué" ou "À valider"

* autorisé à consulter les articles des autres utilisateurs

  • autorisé à gérer les images
  • autorisé à gérer les sections
  • autorisé à gérer les gabarits (pages), les formulaires et les styles CSS
