Diagnostics: Minimum version requirements

Textpattern requires certain functionality present in the PHP programming language and MySQL database server. Please consult the following table to determine the minimum system requirements for the version of Textpattern you have downloaded:

Textpattern version Minimum PHP version Minimum MySQL version
4.6.x 5.3.3 4.1
4.5.x 5.2.3 4.1
4.0.1 - 4.4.1 4.3 3.23

In addition to the above system requirements, the following PHP extensions are required:

  • MySQL (opens an external link in a new window)
  • XML (opens an external link in a new window)
  • JSON (opens an external link in a new window)

Your php.ini file, either global or local, should contain a date.timezone setting.
