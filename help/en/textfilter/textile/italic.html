Textile: Emphasis

Use _underscores_ around text to be emphasized.

The <em> HTML tag is applied:

_a phrase_ → <em>a phrase</em>

If the text is not meant to have structural emphasis, but simply to be styled in italic (e.g. as a design element) use double underscores:

__a phrase__ → <i>a phrase</i>

Note: Quick tags must open and close within a single paragraph to produce valid markup.
