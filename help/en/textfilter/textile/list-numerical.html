Textile: Numeric lists

To make a numeric list, place each item in its own paragraph, preceded by #.

Example

# A first item
# A second item
## A second item first sub item
## A second item second sub item
# A third item
# A fourth item

Will become:

<ol>
    <li>A first item</li>
    <li>A second item
        <ol>
            <li>A second item first sub item</li>
            <li>A second item second sub item</li>
        </ol>
    </li>
    <li>A third item</li>
    <li>A fourth item</li>
</ol>

Which, when rendered by a browser, will appear like this:

  1. A first item
  2. A second item
    1. A second item first sub item
    2. A second item second sub item
  3. A third item
  4. A fourth item

Note: Unmodified paragraphs will be wrapped in <p> tags.
