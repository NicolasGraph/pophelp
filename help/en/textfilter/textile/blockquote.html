Textile: Block quotations

To make an entire paragraph into a block quotation, place bq. before it.

Example

Any old text.

bq. A block quotation.

Any old text.

Will become:

<p>Any old text.</p>
<blockquote>A block quotation.</blockquote>
<p>Any old text.</p>

Which, when rendered by a browser, will appear like this:

Any old text.

A block quotation.

Any old text.

Note: Unmodified paragraphs will be wrapped in <p> tags.
