Preferences: Allow PHP in articles?

When enabled, this setting will allow use of PHP within articles. The author must have sufficient privileges to do so (by default, only ‘Publishers’ (opens an external link in a new window) and ‘Managing Editors’ (opens an external link in a new window) can).

PHP code must have opening and closing PHP tags omitted, and also be enclosed within <txp:php> tags (opens an external link in a new window).
