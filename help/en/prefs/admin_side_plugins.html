Preferences: Use admin-side plugins?

Admin-side plugins can change the operation of the administration interface. They might hook into already existing actions, like editing or publishing articles, or they might add new panels for added/extended functionality.

If you would like to learn more about admin-side plugins, please see the Extending Textpattern (opens an external link in a new window) page in Textpattern User Documentation (opens an external link in a new window). Plugins of all sorts are listed in the Textpattern Resources (opens an external link in a new window) site.
