h1. Write: Override form

If you wish for an article to use a form different from that assigned by default - for example if you want to show only an article excerpt with <a rel="external" target="_blank" href="http://docs.textpattern.io/tags/excerpt"><code><txp:excerpt /></code> <span class="ui-icon ui-icon-extlink">(opens an external link in a new window)</span></a> where you normally show the entire article with <a rel="external" target="_blank" href="http://docs.textpattern.io/tags/body"><code><txp:body /></code> <span class="ui-icon ui-icon-extlink">(opens an external link in a new window)</span></a> - this can be set here.
