<field name="title">CSS Modus</field>
<field name="data_raw">&quot;Cascading Style Sheets&quot;:http://www.w3schools.com/css/default.asp ist eine schlanke, bedeutende Sprache, um das Layout einer Website zentral zu verwalten. Alles, von Schriften und Farben bis hin zu Größen und Positionen, kann durch CSS geregelt werden.
Normalerweise werden CSS Daten in den Kopfbereich &lt;code&gt;head&lt;/code&gt; eines HTML Dokumentes geschrieben. CSS Daten können auch in einer externen Textdatei gespeichert und dann mit dem Dokument verlinkt werden. Moderne Browser verarbeiten diese Daten, speichern sie zwischen und verwenden sie dann für Folgeseiten oder wiederkehrende Aufrufe. Somit kann CSS die Downloadgröße einer Webseite erheblich reduzieren: Das ist nur eine von vielen Verbesserungen der CSS Formatierung gegenüber der alten, herkömmlichen Methode, Websites zu designen.

Textpattern bietet zwei Möglichkeiten, um Style Sheets für  Webseiten zu verwenden.

Im manuellen CSS Modus können Sie jede Style Sheet Information erstellen (oder auch reinkopieren), einfach im Textpattern Interface. Dadurch ist es möglich, Styles schnell und ohne lästiges Hochladen  per FTP  zu bearbeiten.
Style Sheets können mit einfachen Texteditoren oder auch mit WYSIWYG Webeditoren erstellt werden und auch “im ganzen Web&quot;:http://directory.google.com/Top/Computers/Programming/Internet/CSS/Sample_Style_Sheets/Layouts/  gefunden werden. (Anm. des Übers.: der Link bei Google funzt nicht)

Im automatischen CSS Modus wird das Bearbeiten der Styles auf  neuem, höherem Niveau möglich. Ein Interface zum Bearbeiten und eine neue Organisationsmethode stellen  CSS Parameter übersichtlicher und logischer dar.

Im automatischen Modus werden CSS Dateien als ein _layout_ oder ein _style_ klassifiziert. Einzelne Seiten einer Website können sich entweder einen Typ teilen oder auch beide Typen kombinieren.

Layout files definieren die übergeordneten, strukturellen Angaben, wie z.B. Anzahl und Größe der Spalten einer Seite, des Headers oder des Footers.

Style files beeinhalten Angaben zur Schriftart, Farbe und detaillierte Angaben zur Positionierung , die nicht in den Layout files definiert wurden (z.B. Innen- und Aussenabstand und die Abstände einzelner Objekte auf der Seite).</field>

@[alias (optional Titel)]http://url@

Jede auf diese Weise referenzierte URL muss in einer neuen Zeile beginnen.

Beispiel:
Das ist ein &quot;Link&quot;:google zum Google Verzeichnis.

Wenn Sie irgendwo im Dokument diese Zeile haben:
@[google]http://www.google.de/dirhp?hl=de&amp;tab=wd&amp;q=@

... wird der Link ausgegeben als:

Das ist ein &lt;a href=&quot;http://www.google.de/dirhp?hl=de&amp;tab=wd&amp;q=&quot;&gt;Link&lt;/a&gt; zum Google Verzeichnis.



&quot;Weitere Informationen&quot;:http://www.htmlhelp.com/reference/html40/special/a.html</field>
